Sovereignty

Local-First Architecture

Grove is an Express server wrapping an Angular SPA. It reads markdown straight from your filesystem — no database, no sync service, no telemetry. The folder on disk is the source of truth; Grove is just the lens.

  • check_circle Zero external services — your filesystem is the database.
  • check_circle Works offline — it's an Express server on localhost.
  • check_circle MIT licensed — fork, self-host, or ship with your own project.
$ npx grovemd ~/notes
Grove serving /Users/you/notes
→ http://localhost:3000
